安装思路

  1. 进入 GitHub 仓库，先阅读 README、SKILL.md 和许可证。
  2. 整库使用 git clone 下载；单个 Skill 可只复制对应文件夹。
  3. 按你的 Agent 要求放入 ~/.codex/skills、~/.claude/skills 或项目技能目录。
  4. 首次运行前检查脚本权限、外部 API、网络请求与环境变量。

通用下载命令

git clone https://github.com/OWNER/REPOSITORY.git

# Codex 常见目录
mkdir -p ~/.codex/skills

# Claude Code 常见目录
mkdir -p ~/.claude/skills

不同仓库的默认分支、安装目录和依赖可能不同，以仓库最新说明为准。
