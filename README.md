# 2026EAGLE Skills 下载导航

🌐 **在线访问：**  
[https://lqs-github.github.io/2026EAGLE-skills/](https://lqs-github.github.io/2026EAGLE-skills/)

---

## 项目简介

本项目整理了可公开访问和下载的 Agent Skills、科研工具仓库与自动化工作流，按照以下结构进行分类展示：

> 类别 → 名称 → 功能描述 → 推荐场景 → 下载链接

网页支持电脑、平板与手机访问，也欢迎各位转发分享。

---

## 主要功能

- 按类别筛选 Skills
- 支持中文和英文关键词搜索
- 查看 Skill 功能描述与推荐场景
- 一键打开 GitHub 下载页面
- 一键复制 `git clone` 命令
- 导出当前筛选结果为 CSV
- 手机端自适应卡片布局
- 支持通过二维码扫码访问
- 单文件 HTML，可直接部署到 GitHub Pages

## 使用方法

### 在线浏览

直接访问：

[https://lqs-github.github.io/2026EAGLE-skills/](https://lqs-github.github.io/2026EAGLE-skills/)

在网页顶部输入关键词，例如：

```text
论文
PPT
演讲
网页
引用
统计
科研绘图
```

也可以通过分类按钮筛选对应类别。

### 下载 Skill

进入某个 Skill 条目后，点击：

```text
打开下载页
```

即可进入对应的 GitHub 仓库。

网页中的：

```text
复制 Clone
```

按钮可以复制类似下面的命令：

```bash
git clone https://github.com/OWNER/REPOSITORY.git
```

### 常见安装目录

Codex 常见目录：

```bash
mkdir -p ~/.codex/skills
```

Claude Code 常见目录：

```bash
mkdir -p ~/.claude/skills
```

具体安装方式请以各个仓库中的 `README.md` 或 `SKILL.md` 为准。

## 安全提示

本项目主要整理公开 GitHub 仓库和社区 Agent Skills，不代表对所有代码进行过完整安全审计。

安装或运行第三方 Skill 前，建议检查：

- `README.md`
- `SKILL.md`
- `scripts` 目录
- 外部 API 调用
- 网络访问权限
- 环境变量要求
- 软件许可证
- 最近更新时间
- GitHub Issues 与 Pull Requests

不要直接运行来源不明、权限过高或包含可疑脚本的 Skill。
